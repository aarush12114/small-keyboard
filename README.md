# aarush hackpad

aarush hackpad is a 9 key macropad with an OLED Display and uses QMK firmware

## Features:
- case to fit the keyboard
- 128x32 OLED Display
- visiblee XIAO because it looks cool
- 9 Keys

## CAD Model:
Everything fits together using 4 M3 Bolts.

it has 2 pieces 1 on which the pcb sits and another one at top to cover the pcb edges because i wanted to be able to see the pcb a little because it looks cool

## Firmware Overview
This hackpad uses [QMK](https://qmk.fm/) firmware for everything. 

- The 9 keys currently act as macros I dynamically change in VIA.
- The OLED is a cat!! Bongocat!! :3

I might add more in the future! That's it for now

## BOM:
Here should be everything you need to make this hackpad

- 9x Cherry MX Switches
- 9x DSA Keycaps
- 4x M3x5x4 Heatset inserts
- 3x M3x16mm SHCS Bolts
- 2X M3x12mm SHCS Bolts
- 10x 1N4148 DO-35 Diodes.
- 1x 0.91" 128x32 OLED Display
- 1x XIAO RP2040
- 1x Case (2 printed parts)
